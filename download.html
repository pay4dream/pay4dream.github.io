首次打开

首次打开时，系统可能提醒你确认来源。

macOS

使用 ad-hoc 签名，未经过 Apple notarization，不是 Developer ID 签名。

  1. 打开下载的 DMG，把 Pay4Dream 拖进“应用程序”。
  2. 首次打开时右键点 Pay4Dream 图标，选择“打开”。
  3. 如果系统仍然拦截，可以展开下方的备用处理方式。
仍然无法打开时

打开“终端”，运行这行命令后重试：

xattr -dr com.apple.quarantine "/Applications/Pay4Dream.app"

Windows

安装包未签名，SmartScreen 提示是预期现象。

  1. 双击安装包，按安装向导完成安装。
  2. 如果出现系统安全提示，点击“更多信息”。
  3. 选择“仍要运行”，继续启动 Pay4Dream。

开始前准备

你需要邀请码，也需要自己的模型密钥。

Pay4Dream 可连接多种常用模型服务。密钥保存在本机，请确认对应账号有可用额度。
